# Proyecto Final de Métodos Numéricos

Este proyecto implementa técnicas de derivación numérica mediante diferencias finitas y extrapolación de Richardson para aproximar derivadas de funciones de manera eficiente y precisa.

El programa, desarrollado en Java, incluye:

- Diferencias finitas progresivas, regresivas y centradas.
- Cálculo y comparación de errores de aproximación.
- Análisis del efecto del tamaño de paso (h) sobre la precisión.
- Determinación experimental del tamaño de paso óptimo.
- Implementación de la extrapolación de Richardson para mejorar la exactitud de los resultados.
- Construcción de tablas de Richardson para analizar la convergencia de las aproximaciones.

Como caso de estudio se utiliza la función \(f(x)=e^x\), comparando las aproximaciones numéricas con la derivada exacta para evaluar el desempeño de cada método. El proyecto combina fundamentos teóricos de la derivación numérica con su aplicación computacional en problemas científicos e ingenieriles.
